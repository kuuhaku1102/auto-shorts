# auto-shorts

## 目的
本プロジェクトは、ポケモンカードの価格情報をもとに、YouTube Shorts用の解説動画を毎日自動生成する仕組みを構築することを目的としています。

具体的には、まずWebサイトから対象カードの最新価格（例：PSA10価格）を自動で取得します。取得した価格データをもとに、AIを活用して30秒程度のショート動画用台本を自動生成します。その台本をAI音声で読み上げさせ、背景画像とテロップを合成してMP4動画として出力します。

これら一連の工程（スクレイピング、台本生成、音声生成、動画生成）をGitHub Actions上で毎朝自動実行することで、人的作業を最小限に抑えながら、継続的にコンテンツを量産できる状態を作ります。

最終的には、生成された動画をYouTube Shortsに投稿し、広告収益および関連サービスへの導線として活用することを目標としています。

この仕組みの本質は「動画を作ること」ではなく、「動画を自動生成するシステムを構築すること」にあります。人の作業時間に依存せず、データとテンプレートに基づいて継続的にコンテンツを生み出すメディア基盤を作ることが、本取り組みの核心です。
